
# Franzininho WiFi - Get Started 

Bem-Vindo a este repósitorio, aqui estamos nos dedicando ao desenvolvemento de exemplos para a placa Franzininho WiFI com o objetivo de colaborar no processo de aprendizagem da programação e da eletrônica em diversas aplicações.


## Exemplos

Os exemplos propostos possuiem uma abordagem simples e intuitiva para aquelas que estão dando seus primeiros passo nesse ambiente de desenvolvimento tanto para trabalho quanto para estudo. Abaixo está a lista de exemplos propostos:
  * Input - Button
  * Output - Hello World
  * Counter - Contador Up Down
  * PWM - LED fade
